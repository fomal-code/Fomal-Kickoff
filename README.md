## 📖 Overview

Fomal Kickoff is a football news content management system (CMS) developed using Laravel. It provides a clean, responsive interface for readers and a powerful admin dashboard for managing football news articles.

The platform allows administrators to create, edit, publish, and delete football news while authenticated users can browse articles through an intuitive interface.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/fomal-code/fomal-kickoff.git
```

Navigate into the project

```bash
cd fomal-kickoff
```

Install dependencies

```bash
composer install
```

```bash
npm install
```

Create the environment file

```bash
cp .env.example .env
```

Generate the application key

```bash
php artisan key:generate
```

Configure your database inside the `.env` file.

Run migrations

```bash
php artisan migrate
```

Start the development server

```bash
npm run dev
```

```bash
php artisan serve
```

# Fomal Kickoff

Fomal Kickoff is a football news blog built with Laravel. It allows administrators to publish and manage football news articles while providing registered users with a clean and responsive platform to read the latest football updates.

## Features

- User registration and login
- Admin dashboard
- Create, edit and delete articles
- Image upload for news posts
- Responsive design
- About page
- Pagination

## Tech Stack

- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/fomal-code/fomal-kickoff.git
cd fomal-kickoff

composer install
npm install

cp .env.example .env
php artisan key:generate

php artisan migrate

npm run dev
php artisan serve
```
